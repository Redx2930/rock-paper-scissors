# Interest Calculator

A Python-based Interest Calculator that calculates both **Simple Interest** and **Compound Interest** through a simple and user-friendly interface.

This project was built to strengthen Python fundamentals while applying mathematical and financial concepts in a practical way.

## Features

* Simple Interest calculation
* Compound Interest calculation
* User input-based calculations
* Clean and beginner-friendly Python logic
* Fast and lightweight terminal application

## Concepts Used

* Variables and data types
* Mathematical operations
* User input handling
* Conditional statements
* Functions
* Financial formulas

## Formulas Used

### Simple Interest

SI = (P*T*R)/100

### Compound Interest

A = P(1 + r/100)**n

CI = A - P

Where:

* **P** = Principal Amount
* **R** = Rate of Interest
* **T** = Time
