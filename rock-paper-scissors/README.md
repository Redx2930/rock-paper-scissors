# Rock Paper Scissors

A simple Rock Paper Scissors game built using Python.
This project was created to practice Python fundamentals, game logic, condition handling, and user interaction.

## Features

* Interactive command-line gameplay
* Randomized computer choices
* Score/result system
* Beginner-friendly Python project
* Clean and simple game logic

## Tech Stack

* Python 3

## How to Run

1. Clone the repository

```bash
git clone https://github.com/Redx2930/rock-paper-scissors.git
```

2. Open the project folder

```bash
cd rock-paper-scissors
```

3. Run the Python file

```bash
python main.py
```

## Game Rules

* Rock beats Scissors
* Scissors beats Paper
* Paper beats Rock
